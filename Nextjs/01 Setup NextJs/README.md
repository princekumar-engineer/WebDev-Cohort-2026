# Next.js Project Setup

This project is built with Next.js and uses Bun as the package manager and runtime.

## Prerequisites

- Bun installed on your machine

Check installation:

```bash
bun --version
```

If Bun is not installed, follow the official Bun installation guide:

https://bun.sh

---

## Install Dependencies

```bash
bun install
```

---

## Run Development Server

```bash
bun run dev
```

Open:

http://localhost:3000

---

## Build for Production

```bash
bun run build
```

---

## Start Production Server

```bash
bun run start
```

---

## Project Scripts

```json
{
  "scripts": {
    "dev": "bun --bun next dev",
    "build": "bun --bun next build",
    "start": "bun --bun next start",
    "lint": "eslint"
  }
}
```
