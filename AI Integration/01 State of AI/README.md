# State of AI in 2026 – Notes

## 1. What is AI?

**Artificial Intelligence (AI)** is the field of creating machines and software that can perform tasks requiring human intelligence, such as:

* Understanding language
* Learning from data
* Solving problems
* Making decisions
* Recognizing images, speech, and patterns

### Types of AI

1. **Narrow AI (Weak AI)**

   * Designed for specific tasks.
   * Examples: Chatbots, recommendation systems, voice assistants.

2. **General AI (AGI)**

   * Human-level intelligence across many domains.
   * Still not achieved as of 2026.

# 3. What is Generative AI (GenAI)?

**Generative AI (GenAI)** is a type of AI that creates new content based on patterns learned from large datasets.

### Definition

Generative AI produces:

* Text
* Images
* Videos
* Music
* Code
* Designs

instead of only analyzing existing data.

### Examples

* ChatGPT
* AI image generators
* AI coding assistants
* AI video generation tools

### How GenAI Works

1. Collect large datasets.
2. Train neural networks.
3. Learn patterns and relationships.
4. Generate new content when prompted.

### Characteristics

* Creative content generation
* Natural language interaction
* Context awareness
* Personalization
* Automation

# 4. Large Language Models (LLMs)

## What are LLMs?

**Large Language Models (LLMs)** are AI models trained on massive amounts of text data to understand and generate human language.

### Simple Definition

An LLM predicts the most likely next word (or token) based on context.

### Examples of LLM Capabilities

* Answer questions
* Summarize documents
* Translate languages
* Write emails
* Generate code
* Create reports

### Components of LLMs

#### A. Training Data

* Books
* Articles
* Websites
* Research papers
* Documentation

#### B. Parameters

* Internal values learned during training.
* Modern models contain billions or trillions of parameters.

#### C. Transformer Architecture

Most modern LLMs are based on the Transformer architecture introduced in 2017.

### LLM Workflow

1. User enters a prompt.
2. Model understands context.
3. Generates response token by token.
4. Returns output.

### Advantages

* Natural conversations
* Fast content generation
* Knowledge assistance
* Coding support
* Language understanding

### Limitations

* Can hallucinate facts
* Limited reasoning in some situations
* May inherit biases from data
* Requires significant computing resources
