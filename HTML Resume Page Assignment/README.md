# HTML Resume Page Assignment

## Project Description

This project is a single page resume built using only HTML. The goal was to create a clean and structured resume layout using proper semantic tags without using CSS.

---

## Features

Header with name
About section
Skills list
Experience table
Projects section
Education table
Contact details
