# Dev Tool Landing Page — Cursor Inspired

## Project Description

A desktop-first developer tool landing page recreated from the Cursor website. This project focuses on **visual and structural accuracy**, prioritizing layout, typography, and spacing as per the Web Dev Cohort 2026 assignment.

---

## Features

* **Semantic Layout:** Header, Hero, Trusted By, and multi-column Feature sections.
* **UI Components:** Grid-based feature cards, testimonial quotes, and changelog lists.
* **Styling:** Custom CSS variables for the `#14120b` dark theme and **Inter** typography.
* **Accessibility:** Logical $h1 \rightarrow h2 \rightarrow h3$ hierarchy and ARIA landmarks.
* **Pure CSS:** No JavaScript, Tailwind, or external libraries.
