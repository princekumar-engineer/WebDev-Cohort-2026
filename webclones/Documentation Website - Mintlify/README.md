# Documentation Website — Mintlify Inspired

## Project Description

A desktop-first documentation-style landing page recreated from the Mintlify website. This project was built for the **Web Dev Cohort 2026** to practice content structure, sidebar alignment, and high-fidelity layout accuracy using only semantic HTML and CSS.

---

## Features

* **Navigation & Hero:** Top-bar with brand assets and a high-contrast hero section featuring an email CTA.
* **Documentation Preview:** Static sidebar navigation paired with main content cards to mimic a docs interface.
* **Feature Sections:** Two-column highlights and a dedicated "Intelligent Assistant" UI mockup.
* **Enterprise & Social Proof:** Security/compliance blocks and a "Trusted By" logo marquee.
* **Case Studies:** Card-based layout showcasing customer stories with integrated imagery.
* **Footer:** Multi-column link architecture for company and legal information.

## Technical Details

### Fonts & Colors
* **Typography:** Primary sans-serif stack matching Mintlify's clean aesthetic.
* **Colors:** Utilized the official Mintlify palette (sourced via Brandfetch) for backgrounds, primary CTAs, and sidebar borders.

### Constraints & Alignment
| Parameter | Detail |
| :--- | :--- |
| **Tech Stack** | HTML5 + CSS3 Only |
| **Guidelines** | No JavaScript, No Tailwind, No AI-generated code |
| **Layout** | Desktop-only (Fixed width/structure) |
| **Fidelity** | Matches original spacing, font-weight, and component hierarchy |
